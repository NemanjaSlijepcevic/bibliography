# Bibliography Project

A simple web application built with Django and MongoDB for managing bibliographic information. The application allows users to store and manage details about books, including authors, publishers, place of publishing, year of publication, and tags for categorization.

## Features

- Store and manage bibliographic information:
  - Book name
  - Author(s)
  - Publisher
  - Place of publishing
  - Year of publication
  - Tags
- Search and filter entries by tags, author, or year.
- User-friendly interface for adding, updating, and deleting entries.

## Tech Stack

- **Backend:** Django (Python)
- **Database:** sqlite
- **Dependencies Management:** `pip`
